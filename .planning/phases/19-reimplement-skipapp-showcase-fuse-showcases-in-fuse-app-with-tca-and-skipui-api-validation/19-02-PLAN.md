<task type="auto">
  <name>Task 2: Create ShowcaseFeature reducer and ShowcasePath</name>
  <files>examples/fuse-app/Sources/FuseApp/ShowcaseFeature.swift</files>
  <action>
    Create `ShowcaseFeature.swift` containing:

    1. **ShowcasePath** -- a `@Reducer(state: .equatable) enum` with a single case for now:
       ```swift
       @Reducer(state: .equatable)
       enum ShowcasePath {
           case playground(PlaygroundPlaceholderFeature)
       }
       ```
       Where `PlaygroundPlaceholderFeature` is a trivial reducer with `State` containing just the `PlaygroundType` and no actions. This will be expanded in Plans 04-11 as playground reducers are created. Plain view playgrounds and interactive ones alike will initially route through this placeholder.

    2. **PlaygroundPlaceholderFeature** -- minimal reducer:
       ```swift
       @Reducer
       struct PlaygroundPlaceholderFeature {
           @ObservableState
           struct State: Equatable {
               let type: PlaygroundType
           }
           enum Action {}
           var body: some ReducerOf<Self> { EmptyReducer() }
       }
       ```

    3. **ShowcaseFeature** reducer with:
       - `@ObservableState struct State: Equatable` containing `var path = StackState<ShowcasePath.State>()` and `var searchText: String = ""`
       - Computed property `filteredPlaygrounds: [PlaygroundType]` that filters `PlaygroundType.allCases` using word-prefix matching on `searchText` (matching upstream's filter logic)
       - `@CasePathable enum Action` with `case path(StackActionOf<ShowcasePath>)`, `case searchTextChanged(String)`, `case playgroundTapped(PlaygroundType)`
       - Reducer body: `.playgroundTapped` appends `.playground(PlaygroundPlaceholderFeature.State(type: type))` to path; `.searchTextChanged` updates searchText; `.forEach(\.path, action: \.path)` composes ShowcasePath
       - If searchText is empty, return all cases (don't filter)

    4. **ShowcaseView** -- the view for the Showcase tab:
       - Takes `@Bindable var store: StoreOf<ShowcaseFeature>`
       - `NavigationStack(path: $store.scope(state: \.path, action: \.path))` containing a `List` of `ForEach(store.filteredPlaygrounds, id: \.self)` with `NavigationLink(state: ShowcasePath.State.playground(.init(type: playground)))` for each
       - `.navigationTitle("Showcase")`
       - `.searchable(text: $store.searchText.sending(\.searchTextChanged))`
       - `.navigationDestination(for: ShowcasePath.State.self)` using `switch store.case` pattern in the destination closure, rendering a placeholder `Text("Playground: \(type.title)")` for now

    Import ComposableArchitecture and SwiftUI (via SkipFuse).
