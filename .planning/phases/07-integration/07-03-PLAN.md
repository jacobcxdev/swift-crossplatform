Build a comprehensive fuse-app showcase demonstrating every major non-deprecated TCA API and SQLiteData database feature running on both platforms. The app serves as both a reference architecture and an integration test.

R4b also recommends files within FuseApp to avoid Skip/Gradle module explosion.
