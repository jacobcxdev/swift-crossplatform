**Tests to write:**

1. `testSharedBindingProjection` (SHR-05): Create `@Shared(.inMemory("bindTest")) var value = "initial"`. Create `let binding = Binding($value)`. Assert `binding.wrappedValue == "initial"`. Set `binding.wrappedValue = "updated"`. Assert `value == "updated"`. This confirms `$shared` creates a working two-way binding.

2. `testSharedBindingMutationTriggersChange` (SHR-06): Create `@Shared(.inMemory("bindMutate")) var value = 0`. Create `let binding = Binding($value)`. Set `binding.wrappedValue = 42`. Assert `value == 42`. This confirms binding mutations propagate through to the shared state.

3. `testSharedKeypathProjection` (SHR-07): Define a `Codable` struct `struct Parent: Codable, Equatable, Sendable { var child: String = "hello" }`. Create `@Shared(.inMemory("parent")) var parent = Parent()`. Derive child: `let childShared: Shared<String> = $parent.child`. Assert `childShared.wrappedValue == "hello"`. Mutate via child: `childShared.withLock { $0 = "world" }`. Assert `parent.child == "world"`. This confirms keypath projection derives a child Shared.

4. `testSharedOptionalUnwrapping` (SHR-08): Create `@Shared(.inMemory("optTest")) var optional: String? = "present"`. Unwrap: `if let unwrapped = Shared($optional)`. Assert `unwrapped.wrappedValue == "present"`. When nil: set `$optional.withLock { $0 = nil }`. Assert `Shared($optional) == nil`.

5. `testDoubleNotificationPrevention` (SHR-11): Create a `SharedModel` instance (defined above with `@ObservationIgnored @Shared`). Use `withObservationTracking` with an explicit counter to measure notifications precisely:

   ```swift
   let model = SharedModel()
   var observationCount = 0

   // Phase 1: Mutate the @ObservationIgnored @Shared property
   withObservationTracking {
       _ = model.count  // Access to register tracking
   } onChange: {
       observationCount += 1
   }
   model.$count.withLock { $0 += 1 }
   // Assert: observationCount == 0 (ObservationIgnored prevents Observable notification)
   // Assert: model.count == 1 (mutation still applied via @Shared)

   // Phase 2: Mutate the unrelated (non-ignored) property
   withObservationTracking {
       _ = model.unrelated  // Access to register tracking
   } onChange: {
       observationCount += 1
   }
   model.unrelated = "changed"
   // Assert: observationCount == 1 (exactly one Observable notification from unrelated mutation)
   ```

   Done criteria: exactly 0 Observable notifications from @Shared mutation, exactly 1 from unrelated mutation. This proves `@ObservationIgnored` correctly prevents double-notification.

6. `testSharedBindingRapidMutations` (SHR-06, regression): Create `@Shared(.inMemory("rapid")) var value = 0`. Create binding. Set `binding.wrappedValue` in a loop 100 times. Assert final value is 100. No crash, no hang.
