# Plan 15: Fix PeerStore Cleanup Key Mismatch + Add Diagnostic Logging

## Goal

Fix GAP-R3-1 (lazy container deletion causes identity loss for ALL visible items) by normalising cleanup keys. Add diagnostic logging to PeerStore operations to investigate GAP-R3-2 (tab switch peer loss for mixed views).

## Tasks

### Task 1: Fix key type mismatch in ForEach.currentPeerStoreKeys()

**File:** `forks/skip-ui/Sources/SkipUI/SkipUI/Containers/ForEach.swift`

**Change:** In `currentPeerStoreKeys()` (lines 228-246), normalise all key insertions through `composeBundleNormalizedKey()` so cleanup keys match the String keys stored in PeerStore entries.

**Before:**
```swift
private func currentPeerStoreKeys() -> Set<AnyHashable> {
    var keys = Set<AnyHashable>()
    if let indexRange, let identifier {
        for index in indexRange() {
            if let k = identifier(index) { keys.insert(k) }
        }
    } else if let indexRange {
        for index in indexRange() { keys.insert(index) }
    } else if let objects, let identifier {
        for object in objects {
            if let k = identifier(object) { keys.insert(k) }
        }
    } else if let objectsBinding, let identifier {
        let objs = objectsBinding.wrappedValue
        for i in 0..<objs.count {
            if let k = identifier(objs[i]) { keys.insert(k) }
        }
    }
    return keys
}
```

**After:**
```swift
private func currentPeerStoreKeys() -> Set<AnyHashable> {
    var keys = Set<AnyHashable>()
    if let indexRange, let identifier {
        for index in indexRange() {
            if let k = identifier(index) { keys.insert(composeBundleNormalizedKey(for: k)) }
        }
    } else if let indexRange {
        for index in indexRange() { keys.insert(composeBundleNormalizedKey(for: index)) }
    } else if let objects, let identifier {
        for object in objects {
            if let k = identifier(object) { keys.insert(composeBundleNormalizedKey(for: k)) }
        }
    } else if let objectsBinding, let identifier {
        let objs = objectsBinding.wrappedValue
        for i in 0..<objs.count {
            if let k = identifier(objs[i]) { keys.insert(composeBundleNormalizedKey(for: k)) }
        }
    }
    return keys
}
```

**Verification:** Build succeeds. Section 6 delete should only affect the deleted item's peer, not visible neighbours.
