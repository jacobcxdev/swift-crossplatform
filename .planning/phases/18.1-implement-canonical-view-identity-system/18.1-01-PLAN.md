   - If `raw is String || raw is Int || raw is Long` ‚Üí return raw (passthrough for Compose-safe types)
   - If `raw as? AnyOptionalProtocol` succeeds and `unwrappedValue` is non-nil ‚Üí recurse on unwrapped value
   - If `raw as? any Identifiable` ‚Üí recurse on `identifiable.id`
   - If `raw as? any RawRepresentable` ‚Üí recurse on `rawRepresentable.rawValue`
   - Fallback: `return "\(raw)"` (string conversion)

   - `var unwrappedValue: Any? { get }`
   - Conform `Optional` to it: `.some(let w)` returns `(w as? AnyOptionalProtocol)?.unwrappedValue ?? w`, `.none` returns `nil`
   - This replaces the fragile `"Optional(...)"` string stripping in `composeKeyValue()`

3. **`IdentityKeyModifier`** final class extending `RenderModifier`:

   - `@Composable override func Render(content:context:)` ‚Äî transparent: just calls `content.Render(context: context)`

   - `public var selectionTag: Any?` ‚Äî calls `TagModifier.on(content: self, role: .tag)?.value`
