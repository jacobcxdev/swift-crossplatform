## Decisions Made
- Lazy container disposal (GAP-2) is expected Compose behaviour -- `composeBundleNormalizedKey` provides stable diff keys but cannot persist state across disposal. Idiomatic fix is state lifting to parent reducer (standard TCA pattern).
- Tab switch composition scope destruction (GAP-3) is expected Compose behaviour -- `remember` is scoped to composition node. SwiftUI keeps all tabs in hierarchy simultaneously while Compose conditionally renders only selected tab. Future improvement (Skip TabView rendering all tabs) filed as separate concern.
