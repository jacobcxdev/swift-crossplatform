Verification approach:

4. If `skip android test` is not available (no emulator), document this and verify via Robolectric results from Plan 02

**TEST-11 verification (>1000 mutations/sec stability):**

The dedicated stress test (StressTests.swift) achieves 229K mut/sec on macOS but is `#if !SKIP` gated. On Android:
